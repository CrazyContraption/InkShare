InkShare Web Service

Routes:

Get Requests

  • /accounts - Get a list of all accounts, sensitive data is expunged, used for "add friend" auto-complete
  • /canvases - Get a list of all canvases, used for featured canvases
  • /accounts/:id - Get details of an account by direct ID
  • /canvases/:id - Get details of a canvas by direct ID
  • /accounts/friends/:id - Get all friended accounts of a given ID
  • /canvases/participates/:id - Get all participants of a given canvas ID
  • /canvases/viewable/:id - Get all canvases an account's ID may access

Post Requests

  • /accounts/login - Given an email and password, receive an auth token if genuine
  • /accounts/create - Given at least an email, username, password and password confirm, generate a new account
  • /canvases/create - Given at least a name, owner, and participants (must have owner included in participants), generate a new canvas

Put Requests

  • /accounts/update - When ID and valid password of an account is provided through a request body, update all other values provided
  • /canvases/update - When ID of a canvas is provided through a request body, update all other values provided
  • /accounts/delete - When ID and valid password of an account is provided through a request body, delete it
  • /canvases/delete - When ID of a canvas is provided through a request body, mark for deletion
